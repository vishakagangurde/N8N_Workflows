#  N8N Workflows Collection

A curated collection of AI-powered and automation-driven **n8n workflows** designed to solve real-world business problems like revenue leakage, deal risk prediction, sentiment analysis, and market intelligence.

---

##  Workflows Included

###  Deal Escalation Predictor
Predicts deal risk using AI based on CRM activity and triggers alerts.

###  Gold-Silver Ratio Tracker with Opportunity Alerts
Tracks gold vs silver price ratio and identifies trading opportunities.

###  Meeting Prep Bot
Automatically prepares meeting summaries, insights, and action points.

###  Multi-System Revenue Leakage Detector
Detects revenue inconsistencies across multiple systems.

###  News Intelligence Workflow
Analyzes news data to extract insights and trends.

###  Supplier Sentiment Intelligence
Evaluates supplier sentiment using AI from communication or data.

---

##  How to Use

1. Install n8n
2. Import any workflow `.json` file
3. Configure credentials (API keys, DB, etc.)
4. Execute workflow

---

##  Tech Stack

- n8n (Workflow Automation)
- AI APIs (Gemini / OpenAI)
- REST APIs
- JavaScript (Function Nodes)
